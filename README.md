Sonar Bharat – AI-Powered Road & Civic Issue Reporting Platform

Sonar Bharat is a modern, AI-powered web application designed to help citizens report road and civic infrastructure issues across India. 
The platform enables users to capture or upload photos of problems such as potholes, broken roads, garbage dumps, waterlogging, damaged traffic
signals, blocked drains, and street light failures.

The application automatically detects the user's GPS location (with permission) and uses AI-assisted image analysis to identify the issue category,
making the reporting process faster and more accurate. Each complaint is stored with its photo, location, description, and status, allowing users to
track progress from submission to resolution.

The platform also includes an interactive map displaying reported issues, real-time notifications, and an admin dashboard for authorities to review, 
assign, and manage complaints efficiently. Built with a scalable architecture using Next.js, FastAPI, PostgreSQL, and modern web technologies, 
Sonar Bharat aims to improve civic engagement and contribute to safer, cleaner, and better-maintained public infrastructure throughout India.
